# Ativentregas

## Tecnologias utilizadas

- Php 8
- Mysql
- Laravel

## Iniciar aplicação

- Iniciar o xampp
- Iniciar o servidor do laravel - ``` php artisan serve ```
- Iniciar o vite (Javascript) - ``` npm run dev ```
- Acessar a aplicação em - **localhost:8000**
